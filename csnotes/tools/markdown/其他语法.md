## 思维导图

### 使用换行符进行组织
代码声明格式为 \`\`\`mind 。示例：
```
- MarkDown
	- 基本语法
		- 标题与锚点
		标题 #
		- 引用、强调、删除线与分割线
		引用>
		强调 * _
		删除线 ~~
		分割线 ---
		- 链接与图片
		- 列表与表格
		- 代码
	- 绘图
		- 流程图
		- 时序图
		- 甘特图
	- 科学公式
	- 思维导图
```
